Для поднятия сервера на локалхосте  нужно установить XAMPP

После установки запускаем, в нем запускает Apache, mySQL. 

1. Файлы проекта
После этого нужно создать сам сайт на локальном сервере. Он должен лежить в по пути C:\xampp\htdocs\[название сайта] 
Пример Пусть будет сайт блок с кодом внутри 
```
<?php
echo 'Hello World';
?>
```

который мы сохраним как index.php и поместим в папку blog по адресу C:\xampp\htdocs
теперь по localhost/blog/ открывается наш index.php

2. Настройка виртуальных хостов

В случае желания иметь файл проектов по другому пути или иметь переадресацию по произвольному адресу нужно настроить виртуальные хосты
	1. 
	Для этого открывает файл httpd-vhost.conf по адресу C:\xampp\apache\conf\extra
	Добавим в него код для виртуального хоста:
	
	
	<VirtualHost *:80>
	    DocumentRoot "C:\xampp\htdocs"
	    ServerName localhost
	</VirtualHost>
	<VirtualHost *:80>
	    ServerAdmin webmaster@**blog.local**
	    DocumentRoot "C:\my_projects\blog.local\www"
	    ServerName blog.local
	    ServerAlias www.blog.local
	    ErrorLog "C:\my_projects\blog.local\logs\error.log"
	    CustomLog "C:\my_projects\blog.local\logs\access.log" combined
	  <Directory "C:\my_projects\blog.local\www">
	    AllowOverride All
	    Order allow,deny
	    Allow from all
	  </Directory>
		</VirtualHost>
		
		
Здесь каждый <VirtualHost> это редирект, который при вводе ServerName в поле адресной строки читает из директории DocumentRoot

	2.
	После этого нужно настроить файл Hosts по адресу C:\Windows\System32\drivers\etc\hosts
	
	и добавим в него 
	127.0.0.1       blog.local
	127.0.0.1       www.blog.local
	
	Что бы при переходе по адресу  blog.local он спрашивал по ip локальном
	
	После этого перезапускаем апачи и пробуем войти по blog.local

	3. 
	Если ошибка 403 измени в виртуальном хосте
	  <Directory "D:\PHP\blog.local\www">
	        Options Indexes FollowSymLinks MultiViews
	        AllowOverride all
	        Order Deny,Allow
	        Allow from all
	        Require all granted
	  </Directory>
