
### 1 Структура файла
- Код php может находится внутри html кода и должен быть завернут в тег
	<?php
		/// код
	?>
	Или можно создать файл .php и ему достаточно только открывающий тег
- Каждая строчка заканчивается **;**

### 2 Регистры
1. не чувствительны: Ключевые слова, функции, классы 
2. Чувствительны: Имена переменных

### 3 Типы данных
- Boolean
- integer
- float
- string
- Array
- object
- callable

### 4 Переменные

1. Переменная должна начинаться с знака $
2. Переменная должна называться с буквы или нижнего подчеркивания
3. Переменная чувствительна к регистру
4. Тип данных назначается автоматически при объявлении
5. Если переменная начинается с двух $ то, тип переменной становится динамическим

**Константы** объявляеются с помощью команты define(ConstName, Value). Имя константы должно быть строковым( заключено в кавычки)

### 5 Операторы 
#### 5.1 Арифметические операторы


```
+   сложение
-   вычитание
*   умножение
/   деление (всегда возвращает float в PHP 8+ при делении)
//  10 / 3 = 3.333...
%   остаток от деления           // 10 % 3 = 1
**  возведение в степень          // 2 ** 3 = 8
```

#### 5.2 Операторы присваивания



```
=          обычное присваивание
+=         $a += 5    → $a = $a + 5
-=         $a -= 5    → $a = $a - 5
*=         $a *= 5    → $a = $a * 5
/=         $a /= 5    → $a = $a / 5
%=         $a %= 5    → $a = $a % 5
**=        $a **= 3   → $a = $a ** 3
.=         конкатенация строк
           $str .= " world"  → $str = $str . " world"
```

#### 5.3 Операторы сравнения



```
==    равно (с приведением типов)          // 5 == "5" → true
===   строго равно (тип + значение)        // 5 === "5" → false
!=    не равно                             // 5 != "5" → false
!==   строго не равно                      // 5 !== "5" → true
>     больше
<     меньше
>=    больше или равно
<=    меньше или равно
<=>   "космический корабль" (spaceship) 
      возвращает -1, 0 или 1
      // 5 <=> 3 → 1;  3 <=> 3 → 0;  2 <=> 5 → -1
```

#### 5.4 Логические операторы


```
&&    AND (и)            // true && false → false
||    OR (или)           // true || false → true
!     NOT (отрицание)   // !true → false

and   (низкий приоритет, как &&)
or    (низкий приоритет, как ||)
xor   исключающее ИЛИ    // true xor true → false
```

### 7 Циклы

#### 7.1 while


```
while (условие) {
    // код выполняется, пока условие true
}
```

#### 7.2 do...while

```
do {
    // код выполнится ХОТЯ БЫ ОДИН РАЗ
} while (условие);
```
#### 7.3 for

```
for (начало; условие; шаг) {
    // код
}
```
#### 7.4 foreach — самый важный цикл в PHP!

Работает только с массивами и объектами.

```
foreach ($массив as $значение) {
    // $значение — каждый элемент по очереди
}

foreach ($массив as $ключ => $значение) {
    // с ключами
}
```

#### 7.5 Управление циклами

```
break;        // выйти из цикла полностью
break 2;      // выйти из двух вложенных циклов
continue;     // пропустить остаток текущей итерации, перейти к следующей
continue 2;   // пропустить во внешний цикл
```

#### 8. Массивы
Создание массива происходит путем вызова функции array()
	$myArray = array();
При вызове функции без аргументов, она вернет пустой массив, который затем можно заполнить путем присваивания ([1,3,4,5])

Добавить в массив можно или указав индекс:
	$myArray[1] = "21312"
Или в произвольное место, не указав индекс:
	$myArray[] = "123123"

Массивы бывают:
- Индексированными - где каждому значению указан последовательный индекс
- Ассоциативными - где вместо индекса содержится строка
- Многомерными, где массив хранит другие массивы

Размер массива можно узнать вызвав функцию count():
	count($myArray);
