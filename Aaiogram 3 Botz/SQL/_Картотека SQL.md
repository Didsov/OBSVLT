SQL, а в частности SQLAlchemy - это система, которая позволяет обращаться к базе данных как к классам и объектам.
То есть вместо того что бы вызывать sql запрос, мы пишем модель-класс таблицы и в эту модель передаем нужный метод

SQL Alchemy это как раз такой инструмент для работы с этими моделями на питоне.
